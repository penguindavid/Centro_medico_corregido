## Descripcion de la base de datos 

* Registros de pacientes:se guardaran datos como el id_paciente su nombre, appelidos, identificaion del paciente(cedula), fecha de nacimiento,sexo.

* Registro del medico: se guardaran datos como el id_medico, su nombre, apellidos, su identificacion, su fecha de nacimiento y su especialidad, telefono y correo

* Registro de las citas: se guardaran datos como el id, la fecha, hora, la id del paciente, la del medico, id del consultorio...

* consultorio: id consultorio y el nombre(puede ser un numero)

* usuario: id de usuario, login del usuario, estado del usuario, tipo de usuario.
